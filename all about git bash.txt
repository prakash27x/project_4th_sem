Command to create a local git User:
git config --global user.email "youremail@gmail.com"
git config --global user.name "Your Name"

VSCode: https://code.visualstudio.com


\\ Commands to first push your website into GitHub
git init
git add .
git commit -m "firstCommit"
git remote add origin https://github.com/username/repositor...
git push -u origin master


\\ Command to pull the changes from GitHub
git pull


\\ Command to check status of the files
git status


\\ Commands to update the website later
git add .
git commit -m "message"
git push
